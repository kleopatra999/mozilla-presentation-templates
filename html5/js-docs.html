How to help

OK wait, why should I?

  • Preventing doc rotting is a lot of work
  • MDN is not a quick throw-away campaign site
  • Web encyclopedia wiki established in 2005
  • Always been an open community effort
  • Join 600+ monthly MDN wiki editors
  • My upshot: Collaboration on docs makes you smarter!

How to help — writing

  • Write new pages
  • Proof-read pages, provide feedback
  • Perform editorial or technical reviews
  • Rewrite docs for beginners
  • Reuse your blog post
  • Improve our information architecture
  • Tag ALL the docs

How to help — coding

  • Add data to the compat tables, write Browser tests
  • Add example code
  • Add Polyfills and shims
  • Add your code and tell a story in a tutorial
  • Write KumaScript wiki macros to improve wiki automation
  • Hack on Kuma
  • Hack on MDN Firefox add-ons
  • MDN Firefox OS app!?

How to help — even more

  • Translate pages into your language!
    • French and Chinese are actively working on JS doc l10n
  • Link to us
  • Be part of the discussions, join our mailing list
  • Compare the JS docs to other language's docs. What to improve?
  • Follow @newOnMDN for updates
  • Spread the word, give a talk

A JavaScript reference doc has:

  • Short summary first sentence
  • Syntax box
  • A longer description / details
  • Example code
  • References to relevant ES specifications
  • Browser compat table and notes
  • A sidebar with relevant links in this topic
  • Proper tags and flags (Non-standard, experimental, etc)
  • Example page: ES6 Symbol object
